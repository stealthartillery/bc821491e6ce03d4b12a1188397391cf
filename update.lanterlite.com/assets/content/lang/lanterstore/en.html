بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيم

Lanter Store is an application that contains paid digital products and services that can be used to get special features on the Lanterlite application. As an example, the license can open special features such as color choices and type of font on the Lanterlite application.

Lanter Store can be accessed from web store.lanterlite.com. Also can be installed on Mobile with Android OS here or Desktop with Windows OS here.
