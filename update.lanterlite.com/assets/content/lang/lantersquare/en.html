Lanter Square
Lanterlite Social Network

بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيم

Lanter Square is a social network app created by Lanterlite team to facilitate Lanterlite users to communicate with each other using features that follow Islamic laws.

Lanter Square also has a Islamic-lecture schedule feature in certain cities which is displayed every day to make it easier for people to find places to learn about Islam.
