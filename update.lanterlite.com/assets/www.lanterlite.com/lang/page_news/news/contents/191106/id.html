Versi 2.1 Dirilis
Lanter Apps mencapai versi 2.1
Nov 06, 2019

بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيم

Hari ini kami mengumumkan dirilisnya aplikasi Lanterlite versi 2.1.

Pada versi ini, aplikasi Lanterlite telah menerapkan mekanisme enkripsi sendiri untuk kebutuhan sekuritas baik di sisi client maupun server. Selain itu, dari sisi client kecepatan load data semakin membaik, juga disertakan beberapa fitur tambahan seperti fitur petunjuk, dlsb.

Update ini dilakukan untuk semua aplikasi Lanterlite, termasuk Lanter Search dan Lanter Quran. Anda dapat mendownloadnya di Playstore untuk versi mobile android.

Terimakasih.

- Tim Lanterlite
