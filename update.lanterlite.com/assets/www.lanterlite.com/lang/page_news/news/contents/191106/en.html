Version 2.1 Released
Lanter Apps reached version 2.1
Nov 06, 2019

بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيم

Today we announce the release of the Lanterlite application version 2.1.

In this version, Lanter apps has implemented its own encryption mechanism for security needs both client and server. In addition, from the client side the data load speed is getting better, also included several additional features such as guidance feature, etc.

This update is done for all Lanterlite applications, including Lanter Search and Lanter Quran. You can download them on Playstore for the android mobile version.
