Lanter App 2.2
Lanterlite releases all-in-one app.
Nov 22, 2019

بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيم

In Lanter App version 2.2, all Lanterlite apps are combined into one app, the Lanter App. The combined apps include Lanter Site, Lanter Quran, Lanter Search, and Lanter Store. This is done to make easier for users to use features in each Lanterlite app.

Besides, user now is able to access Lanter Store through Lanter App, so the process of purchasing of licenses, as well as other Lanterlite digital goods can be easier.
