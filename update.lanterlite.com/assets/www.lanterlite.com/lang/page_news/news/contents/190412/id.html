Tersedia di Windows Store
Enlite dan Qulite hadir di Desktop.
Apr 12, 2019

Bismillah.

Enlite dan Qulite kini mendukung platform desktop untuk sistem operasi Windows dan telah tersedia di Windows Store. Ini menjadi awal bagi aplikasi-aplikasi Lanterlite untuk dapat mulai berjalan di perangkat-perangkat Laptop dan PC sebagai aplikasi Desktop. Insyaa Allah, secepatnya aplikasi-aplikasi Lanterlite akan mulai merambah ke sistem operasi lain seperti MacOS dan Linux.

Perkembangan fitur juga telah dilakukan pada Enlite dan Qulite, salah satunya yaitu fitur informasi dan pengelolaan akun. Insyaa Allah, Enlite dan Qulite akan terus dikembangkan supaya dapat mempermudah para pengguna dalam memperdalam agama Islam dari aspek apapun.
